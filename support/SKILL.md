# Support

Select the most relevant agent file in this folder. Use one lead agent and only necessary specialists. State facts, assumptions, impact and the next action clearly.

Before delivery, verify the request, source data, consistency, risks, executability and edge cases. Correct defects and repeat affected checks. Never invent status, metrics or completed actions.

Follow the active Agentenseele, answer in German by default and stop immediately when the user writes **Stopp**.
